# 公司深度分析 SKILL

> 视角：站在一个 VC + 战略顾问 + 求职者的混合视角。
> 输出：让用户能回答"这是家什么公司，值得跟它发生什么关系"。

---

## Phase 0：明确分析目的

```
分析一家公司前，请告诉我目的：

A. 投资判断（是否值得投/买股票）
B. 求职判断（是否值得加入）
C. 合作判断（是否值得合作/采购）
D. 竞争判断（如何应对这家对手）
E. 战略研究（为内部 PPT/报告）

不同目的会让我侧重不同维度。
```

### 维度 2：业务结构（必做）

如果是多业务集团，必须画出业务结构图：

```
{母公司}
├── 业务线 A（贡献 X% 收入）
│   ├── 子产品 A1
│   └── 子产品 A2
├── 业务线 B（贡献 Y% 收入）
└── 业务线 C（投资孵化）
```

对每条业务线分析：
- 营收占比
- 增长率
- 利润率
- 在集团内的战略地位（现金牛 / 明星 / 问号 / 瘦狗 - BCG 矩阵）

### 维度 3：财务健康（如有公开数据）

#### 上市公司必填表

| 指标 | 最新值 | 同比 | 趋势 |
|---|---|---|---|
| 营收 | | +/-% | 📈/📉/➡️ |
| 毛利率 | | | |
| 净利润 | | | |
| 现金流 | | | |
| 研发投入占比 | | | |
| Rule of 40（SaaS 公司） | | | |

#### 私有公司用代理指标

- 招聘节奏（猛招 = 资金充足）
- 办公室扩张
- 媒体露出频率
- 高管套现/期权动作

### 维度 4：战略意图（最关键）

**判断战略意图的 4 个信号源**：

1. **CEO 公开言论**（财报会议、访谈、内部信泄露）
   - 近 12 个月反复出现的关键词是什么？
   - 例：黄仁勋反复说"AI infrastructure" = 战略全押 AI

2. **资本动作**
   - 收购了谁 = 缺什么能力
   - 投资了谁 = 看好哪个方向
   - 卖掉了什么 = 放弃哪块

3. **招聘信号**
   - 哪类岗位猛招？（数据科学家 = 加注 AI；BD = 拓展生态；合规 = 应对监管）
   - 关键高管 hire（CTO 来自 Google = 技术转型；CRO 来自 SAP = 企业市场）

4. **专利与论文**
   - 近 1 年专利申请方向
   - 内部研究团队的对外论文

### 维度 5：组织与文化

#### 组织结构特征
- 集中 vs 分布式
- 工程师文化 vs 销售文化 vs 设计文化
- 决策速度（看产品发布节奏可推断）

#### 文化诊断（用代理指标）
- Glassdoor 评分（< 3.5 = 警示）
- 员工任期中位数
- 高管稳定度（CTO/CPO/CMO 近 3 年换了几任）
- 招聘描述里反复出现的关键词

### 维度 6：竞争位置

**4 象限定位**：

|  | 行业地位高 | 行业地位低 |
|---|---|---|
| **增长率高** | ⭐ 当红明星 | 🌱 新兴挑战者 |
| **增长率低** | 💰 现金牛 | 🪦 衰退者 |

判断标准：
- 行业地位 = 市场份额 / 品牌认知 / 客户质量
- 增长率 = 营收 YoY / 用户 YoY

### 维度 7：风险敞口

强制输出风险清单（按概率 × 影响排序）：

| 风险类型 | 具体风险 | 概率 | 影响 | 缓解措施 |
|---|---|---|---|---|
| 监管 | | 🟢/🟡/🔴 | 🟢/🟡/🔴 | |
| 技术 | | | | |
| 竞争 | | | | |
| 财务 | | | | |
| 团队 | | | | |
| 地缘政治 | | | | |

## 数据来源优先级（重要）

按权威性排序，优先用上面的：

### 一级来源（100% 可信）
- 公司财报、招股书、年报
- SEC / 港交所 / 证监会备案
- 公司官方公告

### 二级来源（高可信）
- 知名分析师报告（CB Insights, Gartner, 艾瑞）
- 一线财经媒体（彭博、FT、华尔街日报、晚点 LatePost、36氪深度）
- 公司高管公开访谈

### 三级来源（参考）
- 二线媒体推断、行业人士爆料
- Reddit / 知乎 / 脉脉

### 警示信号
- 公关稿（夸大事实，需打折）
- 自媒体二手转载（信息失真）

每个数据点必须标注来源等级。

## 上市公司额外要做

如果是上市公司，**必须**做：

1. 读最近 2 期财报的 MD&A（管理层讨论与分析）部分
2. 看 10-K / 年报里的 Risk Factors 部分
3. 听最近一次财报电话会议的 Q&A（分析师问的问题暴露真实关切）

---

## 反模式

❌ 把公司官网的"关于我们"页面照抄一遍
❌ 用"行业领导者"、"创新驱动"这类公关词
❌ 忽略风险敞口
❌ 不给"so what"
❌ 财务数据用"很多"、"很少"，必须给具体数字
