# 创始人画像 SKILL

> 哲学：一家公司是创始人的延伸。看懂创始人，就看懂了公司未来的决策。
> 视角：FBI 行为分析师 + 心理学家 + 商业历史学家的混合体。

---

## 触发后先确认范围

```
开始背调前确认：

1. 想了解谁？（确认全名 + 公司，避免重名）
2. 用途？
   A. 准备 1:1 见面 / 面试这家公司
   B. 理解他/她做决策的逻辑（投资 / 合作 / 竞争）
   C. 写人物报道 / 演讲
   D. 单纯学习这位前辈
3. 时间预算？
   A. 15 分钟人物速写
   B. 45 分钟深度画像（推荐）
   C. 2 小时完整人物志（如写文章）
```

## 七维画像框架

### 维度 1：早年经历（塑造世界观）

| 维度 | 内容 |
|---|---|
| 出生年份 / 地点 | |
| 家庭背景 | （创业者/学者/工人/移民 - 这影响价值观） |
| 童年关键事件 | （如果有公开记录） |
| 学校（高中 / 大学 / 研究生） | |
| 专业 | |
| 学术表现 | （辍学者很多 - 是个信号） |

**为什么重要**：早年经历常常决定一个人 30 年后做的决策。Steve Jobs 在 Reed College 的书法课直接催生了 Mac 的字体。

### 维度 2：职业轨迹（能力的积累）

时间线表格：

| 时间段 | 角色 | 公司/项目 | 关键产出/学习 |
|---|---|---|---|
| | | | |

**特别关注**：
- 第一份工作（决定了"原生职场文化")
- 失败的项目（多数大佬都有，关键是怎么 reframe）
- 关键转折（为什么从 A 跳到 B）

### 维度 3：当前公司的创立故事

必填问题：
1. **起源时刻**：什么触发了这次创业？（个人痛点 / 看到机会 / 别人提议）
2. **联合创始人是谁**：互补关系还是相似？
3. **第一笔钱来自哪**：（自有 / 朋友 / 天使 / VC - 不同来源带来不同压力）
4. **早期最关键的 1 个决策**：（招谁 / 转型 / 拒绝什么）
5. **濒死时刻**：（几乎所有公司都有，怎么活过来的）

### 维度 4：决策风格（最值钱的部分）

**4 种风格标签**（多数人是混合型，标出 primary + secondary）：

| 标签 | 特征 | 代表人物 |
|---|---|---|
| 🔬 技术型 | 痴迷技术细节、亲自写代码/做设计、相信工程美学 | Larry Page, 任正非, Vitalik Buterin |
| 🎯 产品型 | 高度共情用户、亲自做 demo、品味驱动 | Steve Jobs, 张小龙, Brian Chesky |
| 📊 数据型 | 一切量化、A/B 测试至上、增长 hacking | Mark Zuckerberg, Brian Halligan |
| 💼 资本型 | 擅长融资 / M&A、组建团队、不亲自做产品 | Masayoshi Son, 王兴（部分） |
| 🌍 远见型 | 押 10 年趋势、忽视近期亏损、宗教式信念 | Elon Musk, Jeff Bezos |
| 🛠️ 运营型 | 体系化、流程化、规模化执行 | Tim Cook, 程维 |

**判断方法**：看他/她在公开场合反复谈论什么 = 真正在乎什么。

### 维度 5：关键引言（3-5 句）

必须收集他/她近 2 年最有代表性的 3-5 句话，每句配：
- 完整原文
- 出处 + 日期
- 上下文说明
- 这句话暴露的世界观

**格式**：

```
> "[完整引言]"
> —— 出处：[访谈/演讲/财报会议]，[日期]
> 上下文：[当时在讨论什么]
> 解读：[这句话告诉我们 ta 真正在乎什么]
```

### 维度 6：重大决策复盘（1-3 个）

挑 1-3 个公开的重大决策，详细复盘：

| 决策时刻 | 选项 | 选了什么 | 当时的反对意见 | 事后效果 | 暴露的决策模式 |
|---|---|---|---|---|---|
| | | | | | |

**示例**：黄铮决定砍掉拼多多的店铺商城功能。所有人反对（"淘宝都有"）。他坚持只做"货找人"。事后效果：差异化护城河。决策模式：极强的第一性原理 + 不怕得罪团队。

### 维度 7：盲点与争议

**任何创始人都有盲点，故意挖出来：**

| 盲点类型 | 该创始人的表现 | 证据 |
|---|---|---|
| 认知盲点 | （比如：技术型 CEO 忽视销售文化） | |
| 性格盲点 | （比如：完美主义导致延期） | |
| 经验盲点 | （比如：toB 出身做不好 toC） | |
| 历史争议 | （股权 / 高管出走 / 公关危机） | |

## 数据收集方法

### 必查的来源

| 来源类型 | 具体平台 | 找什么 |
|---|---|---|
| 公开访谈 | 长视频访谈（不是营销公关稿） | 决策逻辑、世界观 |
| 财报电话会议 | 季度 earnings call | 真实关切（分析师问的问题暴露弱点） |
| 内部信泄露 | 媒体报道的全员信 | 文化、价值观 |
| 社交账号 | Twitter/X、微博、LinkedIn | 即兴反应、品味 |
| 维基百科 | en.wikipedia.org 而非中文版（更完整） | 履历 |
| 自传/传记 | 如果有 | 完整人生轨迹 |
| Glassdoor 评论 | 员工视角的领导风格 | 真实领导力 |
| 早期合作伙伴/员工的访谈 | 出走员工的爆料最真实 | 真实风格 |

### 警惕的来源

⚠️ 公关稿、官方传记（会美化）
⚠️ 营销自媒体的吹捧文章
⚠️ 没有具体出处的二手传闻

## 反模式

❌ **造神**：把人吹成完人。所有创始人都有阴暗面，要写。
❌ **审判**：变成道德批判。客观分析行为模式即可。
❌ **算命**：根据星座、生肖瞎分析。
❌ **复制粘贴**：把维基百科照抄一遍。要给独立的解读。
❌ **缺出处**：每个关键论断都必须有具体引用。

---

## 隐私边界

不调查：
- 配偶 / 子女隐私
- 财产细节（除已公开外）
- 未公开的健康状况
- 任何未公开个人事务

只调查公开的、与商业决策直接相关的信息。
